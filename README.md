# CGPA Calculator Pro
An advanced CGPA calculator with analytics and projections for CUET students.

**⭐Key Features:**
1. Overall CGPA Calculation - Displays cumulative CGPA across all semesters
2. GPA Trend Graph - Interactive line chart showing semester-wise performance
3. Theme Customization - Dark, light, and auto themes with system preference detection
4. Grade Projection - Calculate the grades required to reach a target CGPA
5. Semester Credit Summary - Credit breakdown for each semester
6. Semester Highlights - Automatically detects best and worst performing semesters
7. Copy to Clipboard - Easily copy summary data
8. Download Results as PDF

**⭐Installation Steps:**
1. Go to the [*releases*](https://github.com/RyukNafi/CGPA-Calculator-Pro/releases) tab and download the ZIP file.
2. Extract the contents of the ZIP file anywhere on your PC.
3. Open Chrome and go to the Extensions tab.
4. Enable Developer Mode (top right corner).
5. Drag and drop the extracted folder into the Extensions page.
 
**⭐Usage:**
1. Go to https://course.cuet.ac.bd/
2. Log in with your credentials.
3. Click Published Results.
4. Make sure all of your courses are visible on one page by setting Records per page to 100.
5. Click on the extension icon to view your analytics.
6. Reload if necessary.
